### 📁 Labs/02_Polymorphism
**Тема: Поліморфізм vs Приховування членів.**
* **Virtual/Override**: Дослідив роботу таблиці віртуальних методів (vtable) та інструкцію `callvirt`.
* **New (Shadowing)**: Наочно показав різницю між перевизначенням та приховуванням методу, коли виклик залежить від типу посилання.
* **Sealed Classes**: Застосування модифікатора `sealed` для оптимізації викликів та заборони наслідування.
