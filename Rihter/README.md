# Тема: Ініціалізація структур та .cctor

### Що я дізнався:
1. **CLR Zero-init**: При створенні класу всі його поля-структури занулюються в пам'яті. Конструктор структури при цьому **не викликається**.
2. **this = new Struct()**: У структурах `this` можна перезаписувати. Це швидкий спосіб ініціалізувати всі поля дефолтними значеннями.
3. **.cctor**: Статичний конструктор виконується один раз при першому зверненні до типу.
